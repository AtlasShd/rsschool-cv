### *Task from **«Codewars»** - "N-th Fibonacci"*
Write a function that when given a number (n) returns the n-th number in the Fibonacci Sequence.
```
function nthFibo(n) {
  if (n === 1) {
    return 0;
  } else if (n === 2) {
    return 1;
  } else {
    return nthFibo(n - 1) + nthFibo(n - 2);
  }
}
```
